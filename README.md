# TodoList

TodoList helps you manage your daily tasks efficiently and helps keep a track of it.




## Installation

Install dependencies

```bash
  npm install 
```
Start the project
```bash
  npm start
```
## Used Packages
-  express
-  mongodb
- body-parser
- ejs
